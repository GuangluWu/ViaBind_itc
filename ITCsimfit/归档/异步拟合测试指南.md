## 🐛 故障排除

### 问题1：界面仍然卡死

**可能原因：**
- `future` 或 `promises` 包没有正确加载
- 应用启动时出错

**解决方法：**
```r
# 检查包是否安装
requireNamespace("future", quietly = TRUE)
requireNamespace("promises", quietly = TRUE)

# 如果返回 FALSE，重新安装
install.packages(c("future", "promises"))

# 重启R会话，重新启动应用
```

### 问题2：看到警告信息

如果看到：`"future 或 promises 包未安装，将使用同步拟合模式"`

**解决方法：**
1. 确认包已安装：`installed.packages()[c("future", "promises"), "Version"]`
2. 如果已安装但仍显示警告，重启R会话
3. 重新启动应用

### 问题3：拟合没有结果

**可能原因：**
- 数据问题
- 参数设置问题
- 拟合算法问题

**解决方法：**
- 检查实验数据是否正确导入
- 检查参数范围是否合理
- 尝试不同的拟合算法（L-BFGS-B、LM、DE）
