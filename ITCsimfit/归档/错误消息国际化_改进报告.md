### 2. 创建辅助函数用于格式化错误消息

在 `R/utils.R` 中新增了 `get_error_message()` 函数：

```r
get_error_message <- function(error_code, lang_val = "zh") {
  # 构建翻译键
  title_key <- paste0("error_", error_code, "_title")
  desc_key <- paste0("error_", error_code, "_desc")
  
  # 获取翻译
  title <- tr(title_key, lang_val)
  desc <- tr(desc_key, lang_val)
  
  return(list(title = title, desc = desc))
}
```

**功能特性**：
- 自动构建翻译键
- 支持中英文切换
- 返回结构化的标题和描述
- 错误处理机制（翻译不存在时的降级处理）

## 🎯 实际应用效果

### 示例：文件读取错误

#### 改进前
```r
tryCatch({
  data <- read.csv(file)
}, error = function(e) {
  showNotification(paste("读取文件失败:", e$message), type = "error")
  return(NULL)
})
```

用户看到：`读取文件失败: cannot open the connection`

#### 改进后
```r
tryCatch({
  data <- read.csv(file)
}, error = function(e) {
  handle_error(e,
              context = "读取CSV文件",
              error_code = ERROR_CODES$FILE_READ_ERROR,
              lang_val = lang())
  return(NULL)
})
```

**中文用户看到**：
```

```
