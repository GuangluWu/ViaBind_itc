# 拟合问题快速检查清单

## ❌ 错误信息
**"拟合无法启动：计算出错或模拟与实验数据完全不匹配。"**

---

## 🔍 快速检查清单

按照以下顺序逐一检查：

### ✅ 1. 基本参数检查

- [ ] **实验数据已导入**
  - 检查右上角是否显示数据文件名
  - 检查图表中是否显示实验数据点

- [ ] **实验条件参数已设置**
  - `H_cell_0`（H浓度）> 0
  - `G_syringe`（G浓度）> 0
  - `V_cell`（池体积）> 0
  - `V_inj`（注射体积）> 0
  - `n_inj`（注射次数）> 0

- [ ] **选择了要拟合的参数**
  - 在"拟合参数选择"中至少选择一个参数
  - 常见选择：`logK1`, `H1`, `Offset`, `fH`, `fG`

### ✅ 2. 参数范围检查

- [ ] **logK值在合理范围**
  - 建议范围：1 到 9
  - 如果超出，调整滑条到合理范围

- [ ] **H值在合理范围**
  - 建议范围：-15000 到 5000 (cal/mol)
  - 如果超出，调整滑条到合理范围

- [ ] **修正因子在合理范围**
  - `fH` 和 `fG`：建议 0.5 到 1.5
  - 如果超出，调整到合理范围

- [ ] **Offset在合理范围**
  - 建议范围：-1500 到 1500
  - 用于对齐基线

### ✅ 3. 拟合区间检查

- [ ] **拟合区间在数据范围内**
  - 检查"拟合数据范围"设置
  - 起始针数 ≥ 1
  - 结束针数 ≤ 实验数据总针数
  - 结束针数 ≤ 模拟数据总针数

- [ ] **区间设置合理**
  - 建议至少包含 5-10 个数据点
  - 避免选择异常值较多的区间

### ✅ 4. 模型路径检查

- [ ] **反应路径匹配实验条件**
  - 简单1:1结合：只选择 `rxn_M`
  - 有二聚：选择 `rxn_M` 和 `rxn_D`
  - 有更高聚体：选择相应路径

- [ ] **路径对应的参数已设置**
  - `rxn_M` 需要 `logK1` 和 `H1`
  - `rxn_D` 需要 `logK2` 和 `H2`
  - `rxn_T` 需要 `logK3` 和 `H3`

### ✅ 5. 数据匹配检查

- [ ] **模拟曲线可见**
  - 检查图表中是否显示模拟曲线（红色线）
  - 如果看不到，说明模拟可能失败

- [ ] **曲线大致对齐**
  - 模拟曲线和实验数据应该在同一个数量级
  - 如果差异巨大（如相差100倍），需要调整参数

- [ ] **RSS值显示**
  - 检查右上角的RSS值
  - 如果显示"N/A"，说明数据有问题

## 🛠️ 常见问题及解决方法

### 问题1：模拟计算出错

**症状：**
- 图表中没有模拟曲线
- RSS显示"N/A"

**解决方法：**
1. 检查参数是否在合理范围内
2. 尝试重置参数到默认值
3. 检查反应路径配置是否正确

### 问题2：拟合区间无效

**症状：**
- 错误信息提到"拟合区间无效"

**解决方法：**
1. 检查实验数据有多少针
2. 调整"拟合数据范围"，确保在 [1, 数据针数] 范围内
3. 建议从第2-3针开始，到倒数第2-3针结束

### 问题3：数据完全不匹配

**症状：**
- 错误信息提到"模拟与实验数据完全不匹配"
- 模拟曲线和实验数据差异巨大

**解决方法：**
1. **先手动调整参数对齐曲线**
   - 调整 `Offset` 对齐基线位置
   - 调整 `fH` 和 `fG` 对齐曲线位置
   - 调整 `logK1` 和 `H1` 对齐曲线形状

2. **检查数据单位**
   - 确认实验数据单位是否正确（通常是 cal/mol 或 kcal/mol）
   - 确认浓度单位是否正确（通常是 μM 或 mM）

3. **检查模型是否匹配**
   - 确认选择的反应路径是否匹配实验条件
   - 如果是1:1结合，不要选择二聚路径

### 问题4：参数值异常

**症状：**
- 某些参数值显示为 NaN 或 Inf

**解决方法：**
1. 重置所有参数到默认值
2. 逐步调整参数，观察何时出现异常
3. 避免极端参数值组合

## 📋 推荐的拟合流程

### 步骤1：准备阶段
1. ✅ 导入实验数据
2. ✅ 设置实验条件参数（H浓度、G浓度、体积等）
3. ✅ 选择反应路径（通常从简单的 `rxn_M` 开始）

### 步骤2：手动对齐
1. ✅ 调整 `Offset` 使基线对齐
2. ✅ 调整 `fH` 和 `fG` 使曲线位置对齐
3. ✅ 调整 `logK1` 和 `H1` 使曲线形状大致匹配
4. ✅ 观察RSS值，应该降到合理范围（如 < 1e10）

### 步骤3：开始拟合
1. ✅ 选择要优化的参数（建议先选 `logK1`, `H1`, `Offset`）
2. ✅ 设置拟合区间（建议从第2针到倒数第2针）
3. ✅ 点击"拟合1步"测试
4. ✅ 如果成功，再尝试"完整拟合"

### 步骤4：精细调整
1. ✅ 如果拟合成功但结果不理想，尝试：
   - 调整拟合区间
   - 添加更多参数（如 `fH`, `fG`）
   - 尝试不同的拟合算法（L-BFGS-B, LM, DE）

## 🔧 使用诊断工具

如果以上检查都通过但仍无法启动，可以使用诊断工具：

1. **在RStudio控制台运行：**
   ```r
   source("拟合问题诊断工具.R")
   ```

2. **或者在应用中添加诊断代码**（需要修改server.R）

诊断工具会输出详细的问题分析，帮助定位具体原因。

## 💡 提示

- **先手动对齐，再自动拟合**：这是最可靠的方法
- **从简单开始**：先拟合1:1结合，再考虑复杂模型
- **逐步添加参数**：不要一开始就拟合所有参数
- **检查数据质量**：确保实验数据没有明显异常值
